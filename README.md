# Changelog Builder

Builds a commit based changelog by comparing the svn/git log of two branches.

## Usage

```
cp config.json.dist config.json
node index.js oldBranch newBreanch
```

## install

With [npm](https://npmjs.org) do:

```
npm install changelog-builder
```
